### **Navegación General**
- **Tecla Q**: Cambiar entre escenas.
- **Tecla G**: Acceder directamente a la **escena del juego**.
- **Tecla Z**: Volver a la primera escena desde el juego.

### **Escenas Específicas**
- **Escena 3**: Presiona **1** para crear fuegos artificiales.
- **Escena 4**: Usa las teclas **1 a 4** para visualizar distintos generadores de fuerza.

### **Controles del Juego**
- **Flechas ← / →**: Girar la dirección de disparo del proyectil.
- **Barra espaciadora**: Disparar.
- **Tecla R**: Reiniciar el juego.
- **Tecla N**: Siguiente nivel.
- **Tecla 1**: Activar la zona de viento (prueba).
- **Tecla 2**: Activar la zona de torbellino (prueba).
